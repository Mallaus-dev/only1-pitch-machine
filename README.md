## Deploy to Vercel — 5 Minutes, Zero Cost

### Step 1 — Get your FREE Groq API key
1. Go to console.groq.com
2. Sign up free (no credit card needed)
3. Click "API Keys" → "Create API Key"
4. Copy the key

### Step 2 — Deploy on Vercel
1. Go to vercel.com → sign up free with Google
2. Click "Add New Project"
3. Click "Upload" and drag this entire folder in
4. BEFORE clicking deploy, find "Environment Variables"
5. Add:
   - Name:  GROQ_API_KEY
   - Value: (paste your key here)
6. Click Deploy

### Step 3 — You're live!
Your URL will be: https://only1-pitch-machine.vercel.app
